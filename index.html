WEB이란 무엇인가?

"웹(Web)"은 World Wide Web의 줄임말로, 인터넷을 통해 전 세계의 컴퓨터들이 연결되어 정보를
공유하는 시스템을 말합니다. 웹은 팀 버너스리(Tim Berners-Lee)에 의해 개발되었으며, 그가
1989년에 제안한 개념을 기반으로 하고 있습니다.


웹은 주로 두 가지 핵심 요소로 이루어져 있습니다:

1.웹 페이지(Web Pages): HTML, CSS, JavaScript 등의 웹 기술을 사용하여 작성된 문서로, 텍스트, 이미지,
비디오 등 다양한 형식의 정보를 포함할 수 있습니다. 사용자는 웹 브라우저를 통해 이러한 웹 페이지를 열어볼 수 있습니다.

2.하이퍼텍스트(Hypertext) 및 하이퍼링크(Hyperlink): 웹 페이지 간에 링크를 통해 서로 연결되어 있습니다.
이를 통해 사용자는 원하는 정보로 쉽게 이동할 수 있습니다. 하이퍼링크는 주로 <a>태그를 사용하여 생성됩니다.

웹은 일반적으로 클라이언트-서버 모델을 따르며, 사용자는 웹 브라우저를 통해 웹 서버에 접속하여 웹 페이지를
요청하고 응답을 받습니다. 웹 브라우저는 서버로부터 받은 HTML, CSS, JavaScript 등의 자원을 해석하여
사용자에게 웹 페이지를 표시합니다.


웹은 정보를 전 세계적으로 공유하고, 커뮤니케이션을 촉진하며, 다양한 서비스와 애플리케이션을 제공하는
역할을 합니다. 웹은 전 세계적인 커뮤니케이션과 정보 교환을 위한 강력한 도구로 발전해 왔습니다.
